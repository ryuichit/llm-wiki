# Wei Dong

Corresponding author of the [[webcloak-paper|WebCloak research paper]], leading research on LLM security and web agent defense systems.

## Affiliation

Affiliated with [[nanyang-technological-university|Nanyang Technological University]].

## Research Contributions

### WebCloak Project

As corresponding author, oversaw the comprehensive research on [[llm-driven-web-agents|LLM-driven web agents]] as security threats. The project demonstrated:

- First systematic characterization of LLM scraping threats
- [[webcloak|WebCloak]] defense system reducing scraping success from 88.7% to 0%
- Development of [[llmcrawlbench|LLMCrawlBench]] evaluation framework
- Identification of the [[agent-as-attacker|agent-as-attacker paradigm]]

## Research Interests

- LLM security and safety
- Web security and content protection
- Adversarial defenses against AI systems
- Intelligent agent behavior
