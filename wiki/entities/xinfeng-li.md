# Xinfeng Li

First author of the [[webcloak-paper|WebCloak research paper]], focusing on LLM security and web agent threats.

## Affiliation

Affiliated with [[nanyang-technological-university|Nanyang Technological University]].

## Research Contributions

### WebCloak Project

As first author, led the research on characterizing and mitigating threats from [[llm-driven-web-agents|LLM-driven web agents]] as intelligent scrapers. Key contributions include:

- Development of [[llmcrawlbench|LLMCrawlBench]] benchmark
- Systematic evaluation of 32 scraper variants across three paradigms
- Design of [[webcloak|WebCloak]] defense system
- Identification of [[parse-then-interpret|parse-then-interpret mechanism]] as universal vulnerability

## Research Interests

- LLM security
- Web agent behavior and threats
- Adversarial machine learning
- Content protection systems
