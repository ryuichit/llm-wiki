# Beihang University

**Chinese Name**: 北京航空航天大学 (Běijīng Hángkōng Hángtiān Dàxué)  
**English Name**: Beihang University (Beijing University of Aeronautics and Astronautics)  
**Location**: Beijing, China  
**Founded**: 1952

## Overview

Beihang University is a major research university in Beijing, China, specializing in aeronautics, astronautics, and advanced engineering. The university has expanded to cover a broad range of disciplines including computer science, software engineering, and artificial intelligence.

## School of Software & QRI

**Quantum Research Institute (QRI)** collaboration with School of Software:
- Focus on quantum computing, AI, and software engineering
- Research in LLM-based systems and web automation
- Development of multimodal learning frameworks
- Program synthesis and reinforcement learning applications

## Web Agent Research

### Prune4Web Project (2025)

Lead institution for [[prune4web-paper|Prune4Web: DOM Tree Pruning Programming for Web Agent]]:

**Research Team**:
- [[jing-zhang-beihang|Jing Zhang]]† (corresponding author, professor)
- [[jiayuan-zhang|Jiayuan Zhang]]* (co-first author)
- [[kaiquan-chen|Kaiquan Chen]]* (co-first author)
- Zhihao Lu
- Enshen Zhou
- Qian Yu

**Key Achievements**:
- [[dom-tree-pruning|DOM Tree Pruning Programming]] paradigm (25-50× element reduction)
- [[programmatic-element-filtering|Programmatic Element Filtering]] (88.28% grounding accuracy)
- [[two-turn-dialogue-training|Two-turn dialogue training]] (52.4% Step SR on Cross-Task)
- [[hierarchical-reward-rl|Hierarchical reward RL]] for web agents
- 0.5B model achieving 97.64% recall@20 (better than GPT-4o)

**Technical Innovations**:
- Shift from LLM-based filtering to programmatic pruning
- Multimodal web agent framework (screenshot + DOM)
- Scoring function template with multi-tier weighted matching
- Automated data annotation pipeline (GPT-4o, ~5,000 trajectories)
- Unified model architecture with two-turn dialogue
