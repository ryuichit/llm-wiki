## Contributions

### Web Gatekeeping Analysis

**Research Focus**:
- Large-scale web measurement methodologies
- [[active-blocking|Active blocking]] detection techniques
- Longitudinal analysis of web content control
- Geographic diversity in crawling behavior (7 vantage points)

**Key Insights**:
- Popularity alone doesn't explain blocking behavior
- Even top-ranked misinformation sites remain accessible to AI crawlers
- Reputable sites implement multi-layered defenses (robots.txt + active blocking)
- Misinformation sites show minimal systematic response to AI crawlers

**Technical Contributions**:
- Multi-vantage point crawling infrastructure
- Active blocking measurement methodology
- Validation techniques for crawler behavior analysis
- Statistical analysis of longitudinal trends
