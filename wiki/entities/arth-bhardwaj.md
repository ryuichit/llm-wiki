# Arth Bhardwaj

First author of the [[beyond-beautifulsoup-paper|Beyond BeautifulSoup research paper]], focusing on democratization of web scraping through LLM-powered tools and benchmarking non-expert user capabilities.

## Affiliation

Student at [[saint-francis-high-school|Saint Francis High School]], Mountain View, CA

## Research Contributions

### Beyond BeautifulSoup Project

As first author, led the development of:
- [[novice-web-scraping-benchmark|Benchmark for non-expert web scraping]] spanning 35 websites across 5 difficulty tiers
- [[llm-assisted-scripting|LLM-Assisted Scripting (LAS)]] evaluation framework
- [[end-to-end-llm-agents|End-to-End LLM Agent (ELA)]] evaluation framework
- Unified metrics for measuring both success and ease-of-use
- Systematic comparison of traditional tools vs LLM agents

## Key Findings

Demonstrated that:
- End-to-end agents enable novice users to scrape complex sites with minimal effort (< 5 prompt refinements)
- Traditional tools achieve 82-93% success on simple HTML but completely fail on authentication
- Agents achieve 63-70% success on authentication-protected sites
- Clear division: use traditional tools for static content, agents for protected/authenticated sites
- Web scraping has been democratized - technical barriers dramatically lowered

## Research Interests

- Democratization of technical capabilities through LLMs
- Web scraping accessibility
- Non-expert user capability assessment
- LLM-powered automation
- Practical security implications
